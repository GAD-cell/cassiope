# Inteface web du projet

Cette application web permet d'entrer un PDF et ses sources LaTeX et d'obtenir des statistiques dessus.

## Installation

1. Node JS (Frontend)

    [Installer Node.js](https://nodejs.org/en/download/)

2. Flask (Backend)

    ```bash
    pip install flask
    ```

## Lancer l'application

```bash
./start.sh
```
